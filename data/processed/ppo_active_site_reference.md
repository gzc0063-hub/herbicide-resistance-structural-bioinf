# PPO active-site residue reference hierarchy

**Primary (use for ChimeraX distance-to-active-site normalization):** Heinemann et al. 2007,
*Biochem J* 402:575-580 (tobacco numbering) — 4-residue core: **Arg98, Phe392, Leu356, Leu372**.
Citation (journal/volume/pages) confirmed via Dayan et al. 2010's reference list [6]. The
specific residue-level claims (kcat/Ki effects of mutating each) have **not yet been
independently verified** — cited from the brief, not read directly from Heinemann 2007 itself.

**Secondary/broader (context only, not the primary reference set):** Rangani 2019's 9-residue
table and Hao 2013's 8-residue list — overlapping but not identical to each other or to
Heinemann's core 4. Different methods/purposes across papers, not an error. Don't treat either
as exhaustive.
