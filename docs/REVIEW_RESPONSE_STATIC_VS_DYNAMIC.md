# Review Response: Static Structural Resource vs Dynamic Binding Models

This memo reconciles `Structural Bioinformatics Weed Resistance Review.md` with
the current project state. The review is useful because it names real dynamic
mechanisms that static distance/RSA metrics cannot model. It is not adopted as a
reason to add a molecular dynamics pipeline.

## Project Scope Boundary

This project is a comparative static structural-bioinformatics resource. Its
quantitative claim is that accepted target-site resistance positions occupy
unusual structural zones within each enzyme family, measured by active-site-core
distance percentile, RSA, and conservation.

It is not a binding free-energy predictor, mutant-fold predictor, cross-herbicide
resistance predictor, or molecular dynamics study. Literature MD, docking,
kinetic, and free-energy studies can be cited as mechanism benchmarks where they
exist, but they are not quantities generated by this pipeline.

## Review Points Classified

| Review point | Project response | Status |
|---|---|---|
| Static distance alone cannot explain induced fit, loop motion, solvent effects, or free energy. | Correct. The manuscript should state this explicitly and avoid claims about binding thermodynamics. | Valid limitation |
| Add molecular dynamics / conformational sampling to become predictive. | Not adopted. This would create a different project with much higher validation burden. | Out of scope |
| ALS Pro197/channel effects can be missed by narrow static cores. | Already partly addressed by expanding ALS to an interface-aware 27-residue core; full Pro197 expansion remains future dataset work. | Partly addressed / future work |
| ACCase dimer-interface induced fit matters. | Already addressed structurally by using the 1UYS B+C dimer and ligand-contact core. Dynamic induced-fit energetics remain literature context. | Partly addressed |
| EPSPS Pro106 is not a direct ligand-contact residue and has allosteric/hinge context. | Adopted as mechanism annotation. EPSPS remains directionally consistent but underpowered with one accepted position. | Addressed as caveat |
| PPO loop/helix effects, especially deltaG210, are not reducible to distance alone. | Adopted as the flagship non-core mechanistic exception. | Addressed |
| Raw SASA is not directly comparable across residues/families. | Already fixed by adding Tien-normalized RSA while preserving raw SASA. | Already addressed |
| Static metrics may false-negative second-shell mutations. | Addressed by reporting non-core candidate positions and mechanism annotations rather than only direct-core contacts. | Addressed as analysis framing |
| HPPD should not be forced into TSR-positive analysis. | Already addressed by source audit and contrast-status table. | Already addressed |

## Manuscript Framing to Use

The strongest claim is:

> Across PPO, ALS/AHAS, and ACCase, accepted target-site resistance positions are
> significantly enriched in low within-family distance-to-active-site-core
> percentiles, while mechanism annotation shows that both direct-contact and
> non-core/second-shell resistance positions contribute to this structural zone.

The manuscript should pair that claim with two caveats:

1. Static metrics identify structural context, not binding free energy.
2. Literature mechanisms are needed to interpret why a position confers
   resistance, especially for loop, hinge, channel, and interface mechanisms.
